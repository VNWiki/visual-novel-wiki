# Soreyori no Prologue

## Installation

### Linux / Steam Deck

If you get multiple file, you can merge them into one ISO file:

```
cat file.mdf file.md0 file.md1 file.md2 > SOREYORI.iso
```

You might need to set the `TZ=Asia/Tokyo` environment variable.
