# Soreyori no Prologue

## Installation

### Linux / Steam Deck

If you get multiple files, you can merge them into one ISO:

```
cat file.mdf file.md0 file.md1 file.md2 > SOREYORI.iso
```

You need to set the `TZ=Asia/Tokyo` environment variable for the game to launch.

Use a prefix with wmp11 quartz2 codecs installed with proton-ge for the game to not crash after the opening video.
