# AliceSoft's System

AliceSoft's System engines have been reimplemented and made cross-platform.

Version 3
* [system3-sdl2](https://github.com/kichikuou/system3-sdl2) ([compatibility list](https://github.com/kichikuou/system3-sdl2/blob/master/game_compatibility.md))

Version 3.X
* [xsystem35-sdl2](https://github.com/kichikuou/xsystem35-sdl2) ([compatibility list](https://github.com/kichikuou/xsystem35-sdl2/blob/master/game_compatibility.md))

Version 4
* [xsystem4](https://github.com/nunuhara/xsystem4) ([compatibility list](https://github.com/nunuhara/xsystem4/blob/master/game_compatibility.md))
* [xsystem4-android](https://github.com/kichikuou/xsystem4-android)

Note: "xsystem4" main developer (Nunuhara) also made [ai5-sdl2](https://github.com/nunuhara/ai5-sdl2), which reimplements elf's AI5WIN engine.

## Games

Nunuhara prepared [Appimages for AliceSoft's freewares](https://haniwa.technology/games/). Check out the [page about the xsystem35 setup](https://haniwa.technology/games/preparing-a-game-directory.html) if you want to use "xsystem35" directly.
