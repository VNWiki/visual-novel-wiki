# Kirikiri

## Linux

### Kirikiri SDL2

[Kirikiri SDL2](https://github.com/krkrsdl2/krkrsdl2) is a port of Kirikiri Z to SDL2. It can run on OS like macOS or Linux. [KAG3 for Kirikiri SDL2](https://github.com/krkrsdl2/kag3) is an example project made for this port. Check out [the project website](https://krkrsdl2.github.io/krkrsdl2/en/) and the [Kirikiri2 documentation](https://kirikirikag.sourceforge.net/contents/index.html)

## Android

### Kirikiroid2 Debloated

[Kirikiroid2 Debloated](https://github.com/enaix/Kirikiroid2-debloated) is a recompilation of Kirikiroid2, patched for Android 14 and ads removal.
