# VNDS

[VNDS](https://github.com/BASLQC/vnds) is an open-source visual novel scripting engine, originally made for the Nintendo DS. It was forked to [VNDC](https://github.com/chaoskagami/vndc) but it hasn't been updated in a decade, leaving [VNDS-LOVE](https://github.com/ajusa/VNDS-LOVE) and [vnds](https://github.com/asiekierka/vnds) as the latest forks.

Games need to be specifically ported to this engine to work.
